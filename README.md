## What It Is

* 700KB total size, runs on 512MB RAM
* Works on Chrome/Edge (full WebUSB, Serial, Bluetooth support)
* Control hardware **directly** via WebUSB, Serial, Bluetooth, HID, Modbus RTU
* **Or** add a lightweight backend HAL proxy when needed
* Build apps with HTML/CSS/JS

---

## Hardware Control

**Direct browser control:** USB devices, Serial ports (RS232/485), Bluetooth BLE, HID, Modbus RTU  
**Optional backend:** REST API proxy for complex systems or legacy hardware

Built-in drivers: `webusb`, `serial`, `bluetooth-ble`, `modbus-rtu`, `hid`, `adb`, `storage`, `wakelock`
